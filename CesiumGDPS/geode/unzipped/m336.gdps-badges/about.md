# GDPS Badges
**<cr>ONLY useful on Geometry Dash Private Servers.</c>**

This mod allows <cj>Geometry Dash Private Server</c> owners to <cj>set custom badges</c> depending on <cj>various statistics</c> of players using this mod (<cj>amount of stars</c>, <cj>moons</c>, <cj>coins obtained</c>, <cj>demons beaten</c>...).

**[Check out the GitHub repository for the instructions!](https://github.com/M336G/GDPSBadges#setup-gdps-owners-only)**
